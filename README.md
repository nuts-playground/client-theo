# 테오 플레이그라운드

Javascript로 구현 가능한 게임들을 만들어서 미니게임 모음집

## 진행 상황

![방 입장, 퇴장](./images/screen01.png)
게임 데이터 공유되도록 작업 완료

**우선 작업 필요 내용**

-   불필요한 socket 이벤트가 너무 많다.  
    start, ready, update 등을 하나로 묶어서 게임 방 상태 하나만 관리하도록 바꾸는 게 좋을 것 같다.
